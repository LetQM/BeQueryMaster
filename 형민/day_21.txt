1번 : 자동차 대여 기록에서 대여중 / 대여 가능 여부 구분하기
 - SELECT CAR_ID,
   CASE
       WHEN CAR_ID IN (SELECT CAR_ID FROM CAR_RENTAL_COMPANY_RENTAL_HISTORY
       WHERE START_DATE <= '2022-10-16' AND END_DATE >= '2022-10-16') THEN '대여중'
       ELSE '대여 가능'
   END as AVAILABILITY
   FROM CAR_RENTAL_COMPANY_RENTAL_HISTORY
   GROUP BY CAR_ID
   ORDER BY CAR_ID DESC

2번 : 대여 기록이 존재하는 자동차 리스트 구하기
 - SELECT crcc.CAR_ID FROM CAR_RENTAL_COMPANY_CAR crcc
   INNER JOIN CAR_RENTAL_COMPANY_RENTAL_HISTORY ccrh
   ON crcc.CAR_TYPE = '세단' AND ccrh.CAR_ID = crcc.CAR_ID
   AND month(ccrh.START_DATE) = 10
   GROUP BY crcc.CAR_ID
   ORDER BY crcc.CAR_ID DESC

3번 : 조건에 맞는 사용자와 총 거래금액 조회하기
- SELECT USER_ID,	NICKNAME, SUM(PRICE) as	TOTAL_SALES FROM USED_GOODS_BOARD ugb
  INNER JOIN USED_GOODS_USER ugu
  ON ugb.WRITER_ID = ugu.USER_ID
  WHERE ugb.STATUS = 'DONE'
  GROUP BY WRITER_ID
  HAVING SUM(PRICE) >= 700000
  ORDER BY TOTAL_SALES
