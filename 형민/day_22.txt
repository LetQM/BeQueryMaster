1번 : 조건에 맞는 사용자 정보 조회하기
 - SELECT USER_ID,	NICKNAME, CONCAT(CITY, ' ', STREET_ADDRESS1, ' ', STREET_ADDRESS2) as 전체주소,
   CONCAT(LEFT(TLNO,3), '-', SUBSTRING(TLNO,4,4), '-', RIGHT(TLNO,4))  as 전화번호
   FROM USED_GOODS_BOARD ugb
   INNER JOIN USED_GOODS_USER ugu
   ON ugb.WRITER_ID = ugu.USER_ID
   GROUP BY ugb.WRITER_ID
   HAVING count(ugb.WRITER_ID) >= 3
   ORDER BY USER_ID DESC

2번 : 조회수가 가장 많은 중고거래 게시판의 첨부파일 조회하기
- SELECT CONCAT('/home/grep/src/',u.BOARD_ID, '/', FILE_ID, FILE_NAME,FILE_EXT) as FILE_PATH FROM USED_GOODS_FILE f
  LEFT OUTER JOIN USED_GOODS_BOARD u
  ON u.BOARD_ID = f.BOARD_ID
  WHERE u.BOARD_ID = (SELECT BOARD_ID FROM USED_GOODS_BOARD ORDER BY VIEWS DESC LIMIT 1)
  ORDER BY FILE_ID DESC
